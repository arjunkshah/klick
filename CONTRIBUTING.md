# Contributing

Thanks for helping improve Klick.

## Getting set up

1. Fork the repo and clone your fork.
2. `npm install`
3. Copy `.env.example` to `.env.local` and add Firebase (and optionally `GROQ_API_KEY` for Dex).
4. `npm run dev`

## Pull requests

- Keep changes focused on a single concern when possible.
- Run `npm run lint` and `npm run build` before opening a PR.
- If you change user-visible behavior, a short note in the PR description is enough.

## Security

Do not open public issues for undisclosed security problems. Please report sensitive issues privately to the maintainers (e.g. via GitHub Security Advisories or contact on your profile).
